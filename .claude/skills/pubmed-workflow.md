# PubMed検索ワークフロー

## 検索手順
1. 臨床疑問をPICO形式に整理
   - P (Patient): 対象患者
   - I (Intervention): 介入・検査
   - C (Comparison): 比較対象
   - O (Outcome): 結果・転帰
2. PubMed MCPで検索実行
3. 結果をPMID・DOI付きで記録
4. /docs/pubmed/ に日付付きファイル名で保存

## ファイル命名規則
- 形式: YYYY-MM-DD_キーワード.md
- 例: 2026-03-19_eosinophilic_cellulitis_post_herpes_encephalitis.md

## エビデンスレベル分類
- Level 1: システマティックレビュー・メタアナリシス
- Level 2: ランダム化比較試験（RCT）
- Level 3: 非ランダム化比較試験
- Level 4: コホート研究・症例対照研究
- Level 5: 症例報告・専門家意見

## 推奨度分類（GRADE）
- 強い推奨: エビデンスの質が高い
- 弱い推奨: エビデンスの質が中〜低い
- 推奨なし: エビデンス不十分
