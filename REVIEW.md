# REVIEW.md - PRレビューガイドライン

## 必須チェック項目

### 1. 患者個人情報の漏洩防止
- コード・コメント・データファイルに患者氏名・ID・生年月日等が含まれていないこと
- テストデータやサンプル出力にも実在の患者情報がないこと

### 2. 文献引用の出典明記
- PubMed文献を参照している場合、PMID または DOI が記載されていること
- 出典のない医学的主張がないこと

### 3. KPI計算ロジックの正確性
- 病床稼働率・平均在院日数・回転率の計算式が厚労省「病院報告」の公式定義と一致すること
- 患者フェーズ分類（A群/B群/C群）の日数境界が変更されていないこと

### 4. 日本語の品質
- ユーザー向けテキスト（UI表示・ドキュメント）が日本語であること
- 医学用語は標準的な日本語表記を使用すること

## 注意すべきポイント

### Streamlit アプリ
- Python 3.12 との互換性を確認
- 非推奨API（use_container_width 等）を使用していないこと
- フォント設定がStreamlit Cloud環境で動作すること

### データ出力
- CSV出力のカラム名・形式が既存ファイルと一貫していること
- シミュレーション結果の数値が妥当な範囲内であること

## レビュー対象外
- pubmed-cache/ 配下のキャッシュファイル（自動生成）
- output/ 配下のシミュレーション結果ファイル
