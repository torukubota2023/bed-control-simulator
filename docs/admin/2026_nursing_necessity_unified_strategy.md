## 2026-04-25 Codex反映: アプリ内ギャップ管理

医師別分析タブの「過去1年プロファイル分析」に、**2026看護必要度 ギャップ管理（係数込み）**を追加。

- 厚労省 令和8年度改定資料にある「割合指数 = 該当患者割合 + 救急患者応需係数」をアプリ内で試算
- 過去1年CSVの救急搬送件数から、5F/6Fそれぞれの救急患者応需係数を推計
- 現在の該当患者割合（初期値: 5F 16%、6F 11%）は画面上で修正可能
- 不足ptを「月あたり何該当日を増やす必要があるか」に換算
- C21/C22/C23とA6該当日の月間パッケージで、実行後指数を試算

**重要:** これは医学的に必要な処置・ケアの記録漏れを減らすための管理ツールであり、不必要な処置を増やす提案ではない。

## なぜこれが全員の問題なのか（30秒で理解）

```
基準クリア → 入院料の届出維持 → 病院収入の確保 → 十分な賞与
基準未達  → 入院料ダウン → 年間数千万円の減収 → 賞与に直撃
```

**2026年度改定の公式:**

```
指数 ＝ 該当患者割合 ＋ 救急搬送応需係数 ≧ 19%（必要度I）
         ↑                    ↑
    自分たちで稼ぐ部分        年間220件÷94床×0.005＝1.2%（わずか）
```

救急係数はたった1.2%。**残り17.8%をA項目+C項目で稼ぐしかない。**

## 最大の落とし穴：「稼働率」と「必要度」のジレンマ

```
        ┌─────────────────────────────────────────────┐
        │   該当患者割合 ＝ 該当日数（分子）            │
        │                  ─────────────               │
        │                  全在院日数（分母）            │
        └─────────────────────────────────────────────┘
```

**在院日数を延ばすと → 稼働率は上がる → でも分母が膨張 → 該当割合が下がる（希釈効果）**

| 平均在院日数 | 稼働率 | 該当割合 | 判定 |
|---|---|---|---|
| 16日（現状） | 85.1% | 25.0% | 必要度はクリアだが稼働率不足 |
| 17日（+1日） | 90.4% | 23.5% | 両方ギリギリ達成 |
| 21日（+5日） | 100% | 19.0% | 必要度の崖っぷち |

**単純に在院日数を延ばして稼働率を上げるのは危険。**
→ 「入院初期にスコアを稼ぎ、Day8以降は速やかに退院支援」がベスト戦略。


## 3つの柱：全員で取り組む具体策

### 柱1: A項目で「2点以上」を毎日確保する

**「A2点以上」の達成パターン（覚えてほしい4つ）:**

| パターン | 内容 | 点数 | 対象場面 |
|---|---|---|---|
| 最強パターン | キシロカイン点滴 → A6⑧ | **3点（単独達成）** | ペイン科の日常 |
| 重症パターン | 昇圧剤注射 → A6⑦ | **3点（単独達成）** | 敗血症・ショック |
| 肺炎パターン | 酸素(A2=1) + 注射薬3種(A3=1) | **2点（組合せ達成）** | 内科の日常 |
| 処置パターン | 酸素(A2=1) + 創傷処置(A1=1) | **2点（組合せ達成）** | CV管理中 |

**肺炎患者のタイムライン例:**
```
Day 1-2: 救急搬送後 A7=2点 → 自動的に該当
Day 3-7: 注射薬3種(A3=1) + 酸素(A2=1) = 2点 → 該当維持
Day 8+:  注射終了 → 非該当へ転落 → ここから退院支援加速
```

**注射薬3種のカウント注意:**
- カウントOK: 抗菌薬、H2ブロッカー、制吐剤、etc.
- カウントNG: アミノ酸・糖・電解質・ビタミン（2024年改定で除外済み）
- 上限: 7日間まで

### 柱2: C項目を内科系処置でフル活用する

**2026年改定の最大の追い風:「A2点 又は C1点」で該当**
→ A項目が2点届かなくてもC項目1点あれば該当！

| 区分 | 該当日数 | 主な処置（内科系） | 月間目標件数 | 効果試算 |
|---|---|---|---|---|
| **C21** | **4日間** | CVカテーテル挿入、腰椎穿刺、ERCP、内視鏡止血、CHDF/PMX | 8件 | **+2.7%** |
| **C22** | **2日間** | 気管支鏡/BAL、TEE、EBUS-TBNA | 5件 | **+0.8%** |
| **C23** | **5日間** | PEG、PTCD、CART（胸腹水）、消化管ステント | 2件 | **+0.8%** |
| | | | **合計** | **+4.3%** |

**C23は5日間カウント = 少件数でも効果大。PEG・CARTは見逃し厳禁。**

### 柱3: Day8の崖を意識した戦略的退院支援

```
    ◀──── スコア確保期間 ────▶◀── ここからは希釈 ──▶

    Day1  Day2  Day3  ...  Day7 │ Day8  ...  Day14  Day21
    ███████████████████████████ │ ░░░░░░░░░░░░░░░░░░░░░░
    該当期間（分子に貢献）      │ 非該当期間（分母だけ膨張）
                                │
                           ここで退院支援を加速！
```

**看護部の3つのアクション:**
1. **入院Day3以内:** 退院困難因子スクリーニング（入退院支援加算1の必須要件）
2. **Day7まで:** 老健・サ高住への退院パスを確定
3. **Day8以降:** 在宅復帰率80%要件クリアのため速やかに退院調整

## 稼働率90%を同時に達成する方法

**「ベッドを埋める」思考から「数式をコントロールする」思考へ**

### 収益のスイートスポット = 「緊急入院 × 手術なし」

| | 緊急入院 | 予定入院 |
|---|---|---|
| 手術なし | **入院料最高値（3,367点）** | - |
| 手術あり | 入院料1-1（3,367点） | 入院料1-2（3,267点） |

→ 肺炎・尿路感染症・心不全等の**内科系高齢救急**が、2026年改定で最も有利な患者ミックス

### 稼働率を上げる正しい方法
1. **救急搬送受入の拡大** → 稼働率UP + 救急係数UP + 入院料最高値の一石三鳥
2. **介護施設との協力医療機関協定** → 安定した上り搬送ルート確保
3. **在院日数は19日前後を目標** → 稼働率90%と必要度19%の両立ゾーン

## 毎日のチェックリスト（算定漏れ防止）

### A項目（毎日確認）

- [ ] キシロカイン等の点滴 → A6⑧「抗不整脈剤」**3点**
- [ ] 昇圧剤（ノルアド等）→ A6⑦「昇圧剤」**3点**
- [ ] 抗がん剤注射 → A6① / 麻薬注射 → A6③（各**3点**）
- [ ] 酸素投与中 → A2「呼吸ケア」**1点**（1L/分でも、HFNCでも）
- [ ] NPPV/BiPAP → A2「呼吸ケア」**1点**（夜間のみでも）
- [ ] 注射薬剤3種類以上 → A3 **1点**（栄養系は除外・7日上限）
- [ ] 輸血・血液製剤 → A5 **2点**（見落とし注意）
- [ ] シリンジポンプ管理 → A4 **1点**

### C項目（処置・検査実施時）

- [ ] CV挿入 → **C21（4日間）**
- [ ] 腰椎穿刺・脳脊髄腔注射 → **C21（4日間）**
- [ ] ERCP・内視鏡止血 → **C21（4日間）**
- [ ] 気管支鏡・TEE・EBUS → **C22（2日間）**
- [ ] PEG・PTCD・CART → **C23（5日間）** ← 最長！見逃し厳禁

## 役割分担：全員が「数式のどこ」に貢献するか

| 職種 | 担当する数式の部分 | 具体的にやること |
|---|---|---|
| **医師** | 分子（A/C項目のスコア）| 処置時にC項目該当を意識 / キシロカイン等A項目の活用 / 注射3種のプロトコル設計 |
| **看護師** | 分子（記録）+ 分母（退院支援）| 毎日の必要度評価を正確に記録 / Day3退院困難因子スクリーニング / Day8以降の退院加速 |
| **医事課** | 全体の数式管理 | レセプトコード整合チェック / 週次の該当割合集計 / 救急搬送件数の正確な把握 / コーディング最適化 |
| **管理者** | モニタリング | 週次で病棟別数値確認 / 目標未達時の即時アクション / 科間調整 |
| **副院長** | 戦略統括 | 全体最適化 / 介護施設連携・救急受入拡大の推進 / 経営層への報告 |
