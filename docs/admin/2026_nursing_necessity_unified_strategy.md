## 2026-04-26 Codex追補: 6F実データ版ストラテジーボード

過去1年分析の「看護必要度トレンド（地域包括医療病棟基準）」に、**6F 基準達成ストラテジーボード**を追加。
ミニレクチャーの教育内容を、実データ・6F特性・行動単位へ変換して、医師と看護師が同じ画面で動けるようにした。

### 制度確認

令和8年度改定では、地域包括医療病棟の重症度、医療・看護必要度は **基準該当患者割合に係る指数** で評価される。

```
割合指数 = 該当患者割合 + 救急患者応需係数
```

- 地域包括医療病棟: 必要度I 19%、必要度II 18%
- 救急患者応需係数: 1病床あたり年間救急搬送受入件数 × 0.005
- 救急患者応需係数の上限: 10%

出典:
- 厚生労働省「令和8年度診療報酬改定」地域包括医療病棟資料: `https://www.mhlw.go.jp/content/12400000/001666309.pdf`
- 厚生労働省「重症度、医療・看護必要度における救急患者応需係数と割合指数」: `https://www.mhlw.go.jp/content/12400000/001671032.pdf`

### 6F実績ギャップ

事務提供データ `data/nursing_necessity_2025fy.csv` と `data/past_admissions_2025fy.csv` で確認。
救急搬送 279件 / 94床から、当院仮定の救急患者応需係数は **+1.48%**。

| 評価窓 | 区分 | 実績 | 係数込み割合指数 | 新基準 | 不足 | 月あたり不足患者日 |
|---|---:|---:|---:|---:|---:|---:|
| 12ヶ月平均 | 必要度I | 16.06% | 17.54% | 19% | 1.46pt | 17.5日/月 |
| 12ヶ月平均 | 必要度II | 13.13% | 14.62% | 18% | 3.38pt | 40.7日/月 |
| 直近3ヶ月 | 必要度I | 11.13% | 12.61% | 19% | 6.39pt | 72.0日/月 |
| 直近3ヶ月 | 必要度II | 7.69% | 9.18% | 18% | 8.82pt | 99.4日/月 |

読み方:
- 12ヶ月平均だけを見ると「必要度Iはあと少し」に見えるが、必要度IIは大きく不足している。
- 直近3ヶ月では I/II とも急落しており、2026-06-01 以降の本則適用に向けては **直近3ヶ月の警報を重視**する。
- 不足ptでは現場が動きにくいため、アプリでは「月あたり何患者日足りないか」を最重要表示にした。

### 6Fの病棟特性

過去入院930件の6Fミックス:

| 指標 | 実績 |
|---|---:|
| 内科系 | 82.2% |
| ペイン科 | 13.4% |
| 手術なし | 84.6% |
| 予定入院 | 35.7% |
| 救急車 | 17.0% |
| 中央在院日数 | 8.0日 |

示唆:
- 6Fは「手術でC項目を自然に拾う病棟」ではなく、内科系救急・感染症・呼吸循環管理・処置/検査の記録精度が鍵。
- ペイン科は中央在院日数が長く、適応のあるA6評価と長期在院による分母希釈の両方を見る必要がある。
- 内科系は救急搬送、酸素、注射3種、輸血、シリンジポンプ、C項目の同日評価が主戦場。

### 4つの観点からの解決方針

#### 1. 医療倫理

やること:
- 医学的に必要な治療・ケアを正しく評価し、記録漏れを減らす。
- 迷う症例は医師・看護師・医事で同日確認する。

やらないこと:
- 基準達成目的の不必要な処置
- 虚偽記載
- 病棟都合の患者選別

#### 2. 医学的エビデンス・制度適合

6Fの現実に合う主な確認対象:

| 領域 | 6Fで見るべき候補 | 原則 |
|---|---|---|
| 内科A項目 | 酸素、注射薬3種類、輸血、シリンジポンプ、昇圧薬 | 適応・開始終了時刻・医師指示と看護記録を一致させる |
| ペイン科A6 | 適応が明確な疼痛管理関連処置・薬剤 | 点数目的ではなく、治療実態の評価漏れを防ぐ |
| C項目 | 内視鏡、気管支鏡、腰椎穿刺、ドレナージ、PEG/PTCD/CART等 | 実施日からの該当日数を医事と照合する |
| 分母管理 | Day8以降の非該当長期在院 | 退院支援・転院調整を早期に固定する |

#### 3. 行動人間学

現場行動は「覚えておく」では続かないため、3つの固定習慣にする。

1. 入院時30秒自問: この患者は今日、A2以上・C1以上・救急搬送該当か。
2. 朝ラウンド即答: 看護師からのA項目確認に当日回答する。
3. 木曜6Fハドル: 「不足pt」ではなく「残不足患者日/月」で確認し、責めずに仕組みを直す。

#### 4. UI/視覚効果

アプリに反映した工夫:

- 赤/黄/緑の判定だけでなく、月あたり不足患者日を表示
- 12ヶ月平均と直近3ヶ月を並べ、楽観バイアスを避ける
- 6Fミックスを同じ画面に置き、内科・ペイン科の戦略を切り替えやすくする
- 施策スライダーで「追加該当患者日 → 割合指数」の変化を即時表示
- 医師・看護師・管理者の「今日の一手」を同じ画面に表示

### 当面の運用目標

最低ライン:
- 12ヶ月平均の必要度II不足 40.7患者日/月をまず埋める。

安全ライン:
- 直近3ヶ月の必要度II不足 99.4患者日/月を警報値として、週次で改善傾向を追う。

実務目標:
- 記録・算定漏れ回収: 8患者日/月
- 内科A項目該当: 14患者日/月
- ペイン科A6適応: 6患者日/月
- C項目該当: 12患者日/月
- 合計40患者日/月を初期パッケージとして試算し、未達ならC項目候補と内科A項目の確認頻度を増やす。

## なぜこれが全員の問題なのか（30秒で理解）

```
基準クリア → 入院料の届出維持 → 病院収入の確保 → 十分な賞与
基準未達  → 入院料ダウン → 年間数千万円の減収 → 賞与に直撃
```

**2026年度改定の公式:**

```
指数 ＝ 該当患者割合 ＋ 救急搬送応需係数 ≧ 19%（必要度I）
         ↑                    ↑
    自分たちで稼ぐ部分        年間220件÷94床×0.005＝1.2%（わずか）
```

救急係数はたった1.2%。**残り17.8%をA項目+C項目で稼ぐしかない。**

## 最大の落とし穴：「稼働率」と「必要度」のジレンマ

```
        ┌─────────────────────────────────────────────┐
        │   該当患者割合 ＝ 該当日数（分子）            │
        │                  ─────────────               │
        │                  全在院日数（分母）            │
        └─────────────────────────────────────────────┘
```

**在院日数を延ばすと → 稼働率は上がる → でも分母が膨張 → 該当割合が下がる（希釈効果）**

| 平均在院日数 | 稼働率 | 該当割合 | 判定 |
|---|---|---|---|
| 16日（現状） | 85.1% | 25.0% | 必要度はクリアだが稼働率不足 |
| 17日（+1日） | 90.4% | 23.5% | 両方ギリギリ達成 |
| 21日（+5日） | 100% | 19.0% | 必要度の崖っぷち |

**単純に在院日数を延ばして稼働率を上げるのは危険。**
→ 「入院初期にスコアを稼ぎ、Day8以降は速やかに退院支援」がベスト戦略。

## 3つの柱：全員で取り組む具体策

### 柱1: A項目で「2点以上」を毎日確保する

**「A2点以上」の達成パターン（覚えてほしい4つ）:**

| パターン | 内容 | 点数 | 対象場面 |
|---|---|---|---|
| 最強パターン | キシロカイン点滴 → A6⑧ | **3点（単独達成）** | ペイン科の日常 |
| 重症パターン | 昇圧剤注射 → A6⑦ | **3点（単独達成）** | 敗血症・ショック |
| 肺炎パターン | 酸素(A2=1) + 注射薬3種(A3=1) | **2点（組合せ達成）** | 内科の日常 |
| 処置パターン | 酸素(A2=1) + 創傷処置(A1=1) | **2点（組合せ達成）** | CV管理中 |

**肺炎患者のタイムライン例:**
```
Day 1-2: 救急搬送後 A7=2点 → 自動的に該当
Day 3-7: 注射薬3種(A3=1) + 酸素(A2=1) = 2点 → 該当維持
Day 8+:  注射終了 → 非該当へ転落 → ここから退院支援加速
```

**注射薬3種のカウント注意:**
- カウントOK: 抗菌薬、H2ブロッカー、制吐剤、etc.
- カウントNG: アミノ酸・糖・電解質・ビタミン（2024年改定で除外済み）
- 上限: 7日間まで

### 柱2: C項目を内科系処置でフル活用する

**2026年改定の最大の追い風:「A2点 又は C1点」で該当**
→ A項目が2点届かなくてもC項目1点あれば該当！

| 区分 | 該当日数 | 主な処置（内科系） | 月間目標件数 | 効果試算 |
|---|---|---|---|---|
| **C21** | **4日間** | CVカテーテル挿入、腰椎穿刺、ERCP、内視鏡止血、CHDF/PMX | 8件 | **+2.7%** |
| **C22** | **2日間** | 気管支鏡/BAL、TEE、EBUS-TBNA | 5件 | **+0.8%** |
| **C23** | **5日間** | PEG、PTCD、CART（胸腹水）、消化管ステント | 2件 | **+0.8%** |
| | | | **合計** | **+4.3%** |

**C23は5日間カウント = 少件数でも効果大。PEG・CARTは見逃し厳禁。**

### 柱3: Day8の崖を意識した戦略的退院支援

```
    ◀──── スコア確保期間 ────▶◀── ここからは希釈 ──▶

    Day1  Day2  Day3  ...  Day7 │ Day8  ...  Day14  Day21
    ███████████████████████████ │ ░░░░░░░░░░░░░░░░░░░░░░
    該当期間（分子に貢献）      │ 非該当期間（分母だけ膨張）
                                │
                           ここで退院支援を加速！
```

**看護部の3つのアクション:**
1. **入院Day3以内:** 退院困難因子スクリーニング（入退院支援加算1の必須要件）
2. **Day7まで:** 老健・サ高住への退院パスを確定
3. **Day8以降:** 在宅復帰率80%要件クリアのため速やかに退院調整

## 稼働率90%を同時に達成する方法

**「ベッドを埋める」思考から「数式をコントロールする」思考へ**

### 収益のスイートスポット = 「緊急入院 × 手術なし」

| | 緊急入院 | 予定入院 |
|---|---|---|
| 手術なし | **入院料最高値（3,367点）** | - |
| 手術あり | 入院料1-1（3,367点） | 入院料1-2（3,267点） |

→ 肺炎・尿路感染症・心不全等の**内科系高齢救急**が、2026年改定で最も有利な患者ミックス

### 稼働率を上げる正しい方法
1. **救急搬送受入の拡大** → 稼働率UP + 救急係数UP + 入院料最高値の一石三鳥
2. **介護施設との協力医療機関協定** → 安定した上り搬送ルート確保
3. **在院日数は19日前後を目標** → 稼働率90%と必要度19%の両立ゾーン

## 毎日のチェックリスト（算定漏れ防止）

### A項目（毎日確認）

- [ ] キシロカイン等の点滴 → A6⑧「抗不整脈剤」**3点**
- [ ] 昇圧剤（ノルアド等）→ A6⑦「昇圧剤」**3点**
- [ ] 抗がん剤注射 → A6① / 麻薬注射 → A6③（各**3点**）
- [ ] 酸素投与中 → A2「呼吸ケア」**1点**（1L/分でも、HFNCでも）
- [ ] NPPV/BiPAP → A2「呼吸ケア」**1点**（夜間のみでも）
- [ ] 注射薬剤3種類以上 → A3 **1点**（栄養系は除外・7日上限）
- [ ] 輸血・血液製剤 → A5 **2点**（見落とし注意）
- [ ] シリンジポンプ管理 → A4 **1点**

### C項目（処置・検査実施時）

- [ ] CV挿入 → **C21（4日間）**
- [ ] 腰椎穿刺・脳脊髄腔注射 → **C21（4日間）**
- [ ] ERCP・内視鏡止血 → **C21（4日間）**
- [ ] 気管支鏡・TEE・EBUS → **C22（2日間）**
- [ ] PEG・PTCD・CART → **C23（5日間）** ← 最長！見逃し厳禁

## 役割分担：全員が「数式のどこ」に貢献するか

| 職種 | 担当する数式の部分 | 具体的にやること |
|---|---|---|
| **医師** | 分子（A/C項目のスコア）| 処置時にC項目該当を意識 / キシロカイン等A項目の活用 / 注射3種のプロトコル設計 |

| **副院長** | 戦略統括 | 全体最適化 / 介護施設連携・救急受入拡大の推進 / 経営層への報告 |
