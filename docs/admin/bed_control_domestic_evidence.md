# 国内ベッドコントロール エビデンス集

作成日: 2026-04-17
目的: ベッドコントロールシミュレーター画面「気づきファクト」用の国内データ（カンファ資料で病棟師長・主任・リハ・退院支援NSに提示する考える材料）
対象病棟: 地域包括医療病棟（94床、月間入院約150件、沖縄県那覇市）
収集方針: 2020年以降データ優先、押しつけ調でなく考える材料として機能する形式

---

## カテゴリ1: 厚労省・中医協・病床利用率・平均在院日数（全国平均）

- 2024年の病院全体の病床利用率は 77.0%（前年比+1.4pt）、一般病床 73.3%・療養病床 85.0%（出典: 厚労省 令和6(2024)年 医療施設(動態)調査・病院報告, https://www.mhlw.go.jp/toukei/saikin/hw/iryosd/24/dl/11gaikyou06.pdf）
- 2024年の一般病床の平均在院日数は 15.5日（前年15.7日から0.2日短縮）、全病床平均は25.6日（出典: 同上 厚労省令和6年病院報告）
- 病床利用率の都道府県格差: 2024年最長の高知県 20.2日、最短の愛知県 13.1日と 7.1日の開きがある（出典: GemMed 2026年2月配信, https://gemmed.ghc-j.com/?p=69829）
- 人口10万対病床数は高知県1169.1床 vs 埼玉県525.7床と2.3倍の格差があり、ベッドが多い地域ほどLOSが長い傾向（出典: 同上 GemMed分析）
- 一般病院数は2024年で 7,003施設（前年比62減）、療養病床保有病院は3,338施設で年々縮小傾向（出典: 厚労省令和6年医療施設調査, 表1）

## カテゴリ2: 地域包括医療病棟 の届出・運用実態

- 地域包括医療病棟の届出病院数は 2024年6月22施設 → 10月66施設 → 12月107施設と徐々に増加（出典: 日本経営グループ 病院戦略レポート, https://nkgr.co.jp/useful/hospital-strategy-improvement-115239/）
- 福祉医療機構調査で転換予定は 2.0% にとどまり、78.1%の病院が「転換予定なし」と回答（出典: GemMed 福祉医療機構調査解説, https://gemmed.ghc-j.com/?p=64647）
- 転換断念理由の43.9%が「施設基準のハードルが高い」、在宅復帰率80%以上の達成が最大障壁と指摘（出典: 同上 GemMed福祉医療機構）
- 転換済み病院では入院患者単価が全施設で上昇、経済的メリットは明確（出典: 同上 GemMed福祉医療機構）
- 施設基準は平均在院日数21日以内・救急搬送15%以上・在宅復帰率80%以上・ADL低下5%以内・重症度B 50%以上と多重要件（出典: 地域包括ケア推進病棟協会, https://chiiki-hp.jp/about-hcc）

## カテゴリ3: 病院経営実態（赤字割合・収支）

- 2024年度の医業赤字病院割合は 74.6%（前年70.8%から3.8pt増）、経常赤字病院は 65.6%（前年52.1%から13.5pt増）（出典: GemMed 四病協最終報告, https://gemmed.ghc-j.com/?p=71387）
- 100床あたり医業利益は -1億8044万円、経常利益 -8102万円といずれも前年より悪化（出典: 同上 GemMed四病協最終報告）
- 医業収益は前年比 +2.8% だが支出が +3.5% で増収減益、水道光熱費を含む経費は4.3%増（出典: 同上 GemMed四病協最終報告）
- 帝国データバンクの病院経営動向調査2024年度では赤字構造が常態化し、コスト構造の見直しが急務（出典: 帝国データバンク, https://www.tdb.co.jp/report/industry/20251204-hospital24fy/）
- 調査対象病院は1629施設、2024年度診療報酬改定後も赤字構造は改善せず経営圧迫が継続（出典: 日本病院会 2024年度病院経営定期調査概要版, https://www.hospital.or.jp/pdf/06_20241116_01.pdf）

## カテゴリ4: 救急搬送需要・高齢者増加の将来推計

- 2024年の救急搬送人員は 676.5万人で過去最多、うち65歳以上が 428.2万人（63%）を占める（出典: 第一生命経済研究所 谷口智明レポート, https://www.dlri.co.jp/report/ld/448771.html）
- 高齢者の救急搬送比率は 2010年約51% → 2024年約63% と14年で12pt上昇（出典: 同上 第一生命研レポート）
- 2020年→2040年にかけて75歳以上の救急搬送は36%増、85歳以上は75%増と試算される（出典: 同上 第一生命研レポート）
- 救急搬送件数は2030年ごろ約720万人でピーク、2040年は約710万人と高水準が継続（出典: 同上 第一生命研レポート）
- 地域包括医療病棟は高齢者救急の受け皿として設計されており、5割を急性期、5割を包括期で対応する将来構想（出典: 厚労省 地域医療構想検討会 令和6年9月資料, https://www.mhlw.go.jp/content/10800000/001309842.pdf）

## カテゴリ5: 退院支援・多職種連携の効果データ

- 退院調整看護師(DPN)の全国調査で配置数は2010年461名 → 2014年479名、経験年数2.5→3.1年と徐々に専門性向上（出典: 日本看護科学会誌 2017, https://www.jstage.jst.go.jp/article/jans/37/0/37_37150/_html/-char/ja）
- 退院調整NS参加で「入院7日以内の早期依頼率」が30%→43%に上昇し、支援患者の平均在院日数が短縮（出典: 日本医療マネジメント学会誌 Vol.17(2), https://www.jstage.jst.go.jp/article/jhm/17/2/17_72/_article/-char/ja/）
- 入退院支援カンファレンス導入で脊椎・中枢神経疾患病棟 14.7→13.7日、内科・外科病棟 20.6→19.0日と1日以上の短縮（出典: 看護学雑誌 退院支援研究, https://www.kango-ji.com/journal/download/files/34-1-7.pdf）
- 入退院支援加算を取得した病棟は未取得病棟に比べ平均在院日数が短いことが、急性期・地ケア・回リハ・療養すべてで確認されている（出典: GemMed 入院医療分科会解説, https://gemmed.ghc-j.com/?p=67720）
- 2024年病院看護実態調査: 看護師離職率11.3%(正規)・8.8%(新卒)、夜勤72時間超の看護師が34.3%存在し退院支援業務の余力が限定的（出典: 日本看護協会 2024年病院看護実態調査, https://www.nurse.or.jp/home/assets/20250331_nl1.pdf）
- 約9000名の高齢入院患者分析で「中等度以上の認知症・ADL低下」患者は長期療養施設退院リスクが2.8倍、早期多職種介入が鍵（出典: 東京都健康長寿医療センター, https://gemmed.ghc-j.com/?p=58395）
- 地域一般病棟(13対1・15対1)と療養病棟の3-4割が入退院支援加算を未取得で、支援体制の全国的格差が残存（出典: 同上 GemMed入院医療分科会）

## 補足: 画面表示時の使い方ガイド

- 各ファクトは独立した1行で、回転表示・ランダム表示に対応可能
- すべて出典URLを併記しているため、疑義があれば即座に一次情報を参照できる
- 「押しつけ」ではなく「考える材料」として機能するよう、複数視点(全国平均/都道府県格差/将来推計/現場実態)を混在させている
- カテゴリ4(救急搬送需要増)とカテゴリ2(地域包括医療病棟)は、自院の進路判断の根拠として特に有用
- カテゴリ3(赤字構造)は「稼働率を上げるだけでは解決しない」という経営認識の形成に活用可能
