# ポータブルブラウザ運用ガイド — ベッドコントロールシミュレーター

## 1. ポータブル Firefox とは

ポータブル Firefox（Firefox Portable Edition）は、通常の Firefox と同じ機能を持つブラウザだが、以下の特徴がある:

- **インストール不要** — フォルダをコピーするだけで使える
- **レジストリ変更なし** — Windows のシステム設定を一切変更しない
- **電子カルテへの影響ゼロ** — 既存の Edge や他のアプリケーションに干渉しない
- **持ち運び可能** — USBメモリに入れて別の端末でも使える
- **管理者権限不要** — 一般ユーザー権限で実行可能

**院内の Edge はバージョン90に固定されており電子カルテ用として維持し、ベッドコントロールシミュレーターにはこのポータブル Firefox を専用ブラウザとして使用する。**

## 2. 入手方法

### PortableApps.com からダウンロード

1. インターネット接続のあるPCで以下のURLにアクセスする:
   - https://portableapps.com/apps/internet/firefox_portable

2. 「Download」ボタンをクリックし、インストーラーをダウンロードする
   - ファイル名例: `FirefoxPortable_XXX.paf.exe`（XXXはバージョン番号）

3. ダウンロードしたファイルを実行し、展開先フォルダを指定する
   - 推奨展開先: `C:\PortableApps\FirefoxPortable\`
   - または共有フォルダ: `\\server\shared\PortableApps\FirefoxPortable\`

4. 展開完了後、`FirefoxPortable.exe` が生成される

## 3. 配布方法

### 方法A: 共有フォルダ経由（推奨）

1. 院内ファイルサーバーの共有フォルダに配置する:
   ```
   \\server\shared\PortableApps\FirefoxPortable\
   ```

2. 各端末からはこの共有フォルダ上の `FirefoxPortable.exe` を直接実行する
   - または、後述のショートカットを作成する

3. **メリット**: 一箇所で管理でき、アップデートも1回で済む

### 方法B: USBメモリ経由

1. USBメモリに FirefoxPortable フォルダをコピーする

2. 各端末の以下のいずれかにコピーする:
   - `C:\PortableApps\FirefoxPortable\`
   - `%USERPROFILE%\Desktop\FirefoxPortable\`（デスクトップ）

3. **メリット**: ネットワーク障害時でもローカルで動作する

### 方法C: 両方併用（最も確実）

- 通常は共有フォルダから実行
- バックアップとしてローカルにもコピーを配置
- `deploy/launch_portable_firefox.bat` は自動的に両方を探索する

## 4. 端末での使い方手順

### 初回セットアップ

1. デスクトップに配置された「ベッドコントロール」ショートカットをダブルクリックする
   - または `deploy/launch_portable_firefox.bat` を実行する

2. ポータブル Firefox が起動し、自動的にシミュレーターのURLが開く

3. パスワードを入力してログインする

### 日常の使い方

1. デスクトップの「ベッドコントロール」ショートカットをダブルクリック
2. パスワードを入力
3. ダッシュボードが表示される
4. 使い終わったらブラウザを閉じる（サーバーは動作し続ける）

**注意: Edge（電子カルテ用）とポータブル Firefox（シミュレーター用）は同時に使用可能。**

## 5. ショートカット作成方法

### 手動でショートカットを作成する場合

1. デスクトップを右クリック → 「新規作成」→「ショートカット」

2. 以下のいずれかをリンク先に入力する:

   **共有フォルダから実行する場合:**
   ```
   "\\server\shared\PortableApps\FirefoxPortable\FirefoxPortable.exe" http://192.168.1.100:8501
   ```

   **ローカルコピーから実行する場合:**
   ```
   "C:\PortableApps\FirefoxPortable\FirefoxPortable.exe" http://192.168.1.100:8501
   ```

3. ショートカット名を「ベッドコントロール」に設定する

4. （任意）アイコンを変更する場合:
   - ショートカットを右クリック → プロパティ → アイコンの変更

### バッチファイルを使う場合

`deploy/launch_portable_firefox.bat` をデスクトップにコピーすれば、自動的にポータブル Firefox を探索して起動する。サーバーURLの変更は `.bat` ファイル内の `SERVER_URL` を編集する。

## 6. FAQ

### Q: セキュリティ上の問題はないか?
**A:** ポータブル Firefox は Mozilla 公式ビルドをベースにしており、通常の Firefox と同等のセキュリティを持つ。院内LANのみで使用し、インターネットアクセスは不要であるため、外部からの脅威は限定的。IT部門の承認を得た上で導入すること。

### Q: Firefox のアップデートはどうするか?
**A:** PortableApps.com から最新版をダウンロードし、既存のフォルダに上書き展開する。共有フォルダ方式であれば、サーバー上の1箇所を更新するだけで全端末に反映される。頻繁なアップデートは不要だが、半年に1回程度の更新を推奨する。

### Q: 複数端末で同時に使えるか?
**A:** はい。ポータブル Firefox はそれぞれの端末で独立して動作する。ただし共有フォルダから直接実行する場合、同時実行数に制限がある場合がある。その場合はローカルコピー方式を併用する。

### Q: Edge と Firefox を同時に使って問題ないか?
**A:** 全く問題ない。Edge は電子カルテ用、Firefox はシミュレーター用として完全に独立して動作する。ポータブル Firefox は Edge の設定やデータに一切干渉しない。

### Q: ブックマークや設定はどこに保存されるか?
**A:** ポータブル Firefox のフォルダ内（`Data\profile\` サブフォルダ）に保存される。レジストリやユーザープロファイルフォルダには書き込まない。

### Q: 動作が遅い場合はどうすればよいか?
**A:** 共有フォルダから直接実行している場合、ネットワーク速度の影響を受ける。ローカルコピーに切り替えると改善する場合がある。

## 7. IT部門確認事項

導入前に IT 部門（SE）へ確認・承認を得るべき事項:

- [ ] ポータブルアプリケーションの端末への配置に関するセキュリティポリシー上の問題がないか
- [ ] 端末のセキュリティソフト（ウイルス対策等）がポータブル Firefox の実行をブロックしないか
- [ ] 共有フォルダへの配置が可能か、アクセス権限の設定方法
- [ ] 端末のローカルディスクへのコピーが許可されるか
- [ ] Firefox Portable のバージョン管理・アップデート方針
- [ ] ネットワーク監視ツールとの互換性（ポータブル Firefox の通信が制限されないか）
