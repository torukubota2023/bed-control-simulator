## 救急患者応需係数の計算方法

### 基本式

```
看護必要度 該当患者割合（判定用指数）
  = 該当患者割合 + 救急患者応需係数

救急患者応需係数（小数）
  = 病床あたり年間救急搬送件数 × 0.005
  
ただし上限 10%（0.10）
```

### 「病床あたり年間救急搬送件数」の詳細

```
病床あたり年間救急搬送件数
  = 当該医療機関全体の直近 1 年間の救急搬送受入件数
  × 救急搬送患者のうち該当入院基本料算定病床に入院した割合
  ÷ 該当入院基本料算定病床数
```

**当院ケース**:
- 当該医療機関全体の救急搬送受入件数 = **279 件/年**（過去 1 年実データ）
- 救急患者のうち地域包括医療病棟に入院した割合 ≒ **100%**（当院は他病棟ほぼなし）
- 該当病床数 = **94 床**（5F 47 + 6F 47）
- → 病床あたり = 279 / 94 = **2.97 件/床/年**
- → 応需係数 = 2.97 × 0.005 = **0.01484 ≒ 1.48%**

### 救急搬送受入件数の定義

- **救急用自動車・救急医療用ヘリコプター**により搬送された患者を受け入れた件数
- 自院内転棟、夜間休日緊急搬送医学管理料の対象例も考慮（疑義解釈で具体例提示）
- 直近 1 年間 = 報告月（4 月など）の前年同月から起算

### 上限の意味

- 計算結果が 10% を超える場合 → 加算は **10%（0.10）が上限**
- 当院は 1.48% で上限の 1/7 程度 → 救急受入件数を増やせば余地あり
