## 各項目の判定基準（重要ポイントのみ）

### A1 創傷処置（褥瘡を除く）

- **創傷の定義**: 皮膚・粘膜の破綻
  - **含む**: 縫合創、潰瘍、蜂窩織炎、熱傷、化学的損傷
  - **除く**: 単純な穿刺創（採血、注射、点滴の刺入部のみなど）
- **対象処置**:
  - 洗浄、消毒、止血
  - 薬剤の塗布・注入
  - ガーゼ・フィルム材等の創傷被覆材の貼付・交換
- **対象外**:
  - 診察・観察のみ
  - ガーゼを剥がすのみ
  - VAC 療法（陰圧閉鎖療法、別評価）
  - 眼科手術後の点眼
  - ストーマ排泄物処理

### A2 呼吸ケア

- **対象**: ネブライザー、体位ドレナージ、口腔・鼻腔吸引（喀痰吸引のみは除外）
- **除外**: 喀痰吸引のみ、人工呼吸器管理（別項目）、酸素吸入のみ

### A3 注射薬剤 3 種類以上の管理

- **令和 8 改定で旧「点滴ライン同時 3 本以上」から名称・概念変更**
- **最大 7 日間連続**でカウント可能
- 同時に 3 種類以上の注射薬剤（抗菌薬、輸液、電解質補正等）を投与している場合
- 単独投与しても 3 種類使用していれば該当

### A6 専門的な治療・処置の判定

- 各サブ項目独立評価。複数該当は加算ではなく**最高得点を採用**
  - 例: ③ 麻薬注射 (3点) + ⑨ ヘパリン持続 (3点) → A6 = 3点（加算ではない）
- 「注射剤のみ」とある項目は内服・貼付は対象外
- 「持続点滴」とある項目は単発静注は対象外

### A7 救急搬送後の入院

- **令和 8 改定で 5 日間 → 2 日間に短縮**
- 救急搬送日（入院日）から 2 日間カウント
- Ⅰ: 急性期一般入院料 系
- Ⅱ: 地域包括ケア病棟、地域包括医療病棟 等

## 当院運用上のヒント

### 取りこぼし防止のチェックポイント

1. **A1 創傷処置**: 蜂窩織炎、糖尿病性足病変は意外と日常的。チェック忘れ多い
2. **A3 注射薬剤 3 種類以上**: 内科で頻発するが、「3 種類同時」を意識した記録が必要
3. **A6③ 麻薬持続点滴**: 急性膵炎、胆石症、がん性疼痛で経口移行を急ぎすぎない
4. **A6⑦ 昇圧剤**: 敗血症、心不全急性増悪で短時間でも使用すれば該当
5. **A6⑧ 抗不整脈剤**: アミオダロン、リドカイン点滴。ペイン科 PHN 治療も対象
