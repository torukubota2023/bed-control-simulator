## 📊 各タブの使い方

### タブ 1「📊 今週の需要予測」

毎朝、**出勤時に 30 秒だけ見てください**。

| 見るべきポイント | 確認内容 |
|----------------|---------|
| **今週の総合判定バナー** | 🟢 通常 / 🟡 標準 / 🔴 高需要 で色分け |
| **7 日間予測テーブル** | 今日から日曜までの曜日別予想入院・空床 |
| **大型連休カウントダウン** | 次の連休まで何日、退院調整開始推奨日 |

→ これだけで「**今週は退院調整を強めに / 通常運用でよし**」が判断できます。

### タブ 2「📋 退院候補リスト」

**退院調整会議（毎週水曜）で開いて画面共有してください**。

**使い方:**
1. 現在入院中の患者さんが **A / B / C 区分** で自動仕分けされています
   - 🟢 **A: 連休前退院候補** — 家族に打診してみる価値あり
   - 🟡 **B: 連休中継続妥当** — そのまま継続
   - 🔵 **C: 連休明け退院予定** — そのまま
2. **🟢 A 区分の各患者** について、主治医・MSW で退院可否を議論
3. 議論結果に応じて、画面上の **「手動区分」欄** を変更
4. 会議終了時に **「📄 CSV ダウンロード」** をクリックして会議記録として保存

**重要な注意:**
- A/B/C の自動判定は **近似値** です。必ず臨床的評価を優先してください
- 本リストには患者氏名・生年月日・診断名は含まれていません（個人情報保護のため）

### タブ 3「📅 予約可能枠」

**予約受付事務員の皆様が、新規予約を受ける時に確認してください**。

**使い方:**
1. カレンダーで新規予約の希望日を探す
2. セルの色を見て判断:
   - 🟢 **通常（5 枠以上空き）** → そのまま予約確定
   - 🟡 **やや混雑（2〜4 枠空き）** → 代替日を提案（近隣の 🟢 の日）
   - 🔴 **満員（0〜1 枠）** → 必ず代替日を提案、または主治医と相談
   - 🔵 **連休中** → 連休明け以降を提案
3. クリックすると右側に詳細パネルが表示されます（予想入院・既予約・推奨運用）

## 📝 フィードバックのお願い

以下の点について、試験運用中にお気づきになったことを **いつでも副院長までご連絡** ください:

### 特にお聞きしたいこと

1. **判定の的中率**:
   - A 区分と判定された患者が、実際に連休前に退院できましたか？

## ⚠ 試験運用中の注意事項

### ✅ 推奨事項

- **毎朝 30 秒、タブ 1 を見る習慣** を付けてください
- 退院調整会議で **タブ 2** を画面共有（プロジェクター or 大型モニター）
- **予約受付時は必ず タブ 3** を確認してから判断

### ❌ してはいけないこと

- **画面の判定結果を絶対視しないでください**。判定は近似値です
- 医学的判断よりダッシュボードを優先することは**絶対にやめてください**
- 本ツールは **意思決定の補助** であり、**意思決定そのものではありません**

### 🚨 トラブル時

- ブラウザを閉じて再アクセスしてください
- それでも解消しない場合、副院長までご連絡ください
- 院内情報システム部門ではなく、副院長に直接ご連絡いただくのが早いです（このツールは副院長内製のため）

## 📈 試験運用の「成功」の定義

試験運用期間終了時点で、以下のいずれかが確認できれば成功です:

1. **GW 期間中の平均空床数** が 2025 年比で 1 床以上改善
2. **GW 明け水曜（5/6）** の受入が混乱なく行われた
3. **ベッドコントロール担当者**から **「このダッシュボードがあると運用しやすい」** という肯定的フィードバックがある
4. **理事会（5 月下旬予定）**で、**副院長が「実際に使ってみた担当者の声」を引用できる**

試験運用の結果が失敗であっても、それはそれで貴重な学びです。**率直なご意見をぜひお聞かせください**。
